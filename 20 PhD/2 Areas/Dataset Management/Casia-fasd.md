# CASIA-MFSD

- **Year:** 2012
- **Subjects:** 50
- **Live/Spoof:** 150/450 (Video)
- **Setup:** 7 scenarios and 3 image quality
- **Attack Types:** Print(flat, wrapped, cut), Replay(tablet)
- **Link:** [Paper](http://www.cbsr.ia.ac.cn/users/jjyan/ZHANG-ICB2012.pdf)

## Description
One of the early benchmark datasets for face anti-spoofing.
